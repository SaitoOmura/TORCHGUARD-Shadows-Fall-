# TORCHGUARD-Shadows-Fall-
ユニットを配置して敵の進行を防ぐタワーディフェンスゲームです。

## 使用技術
- C++
- DxLib

## 制作人数
4人

## 担当範囲
※チーム制作のため、自分が担当した部分のみ記載しています

- 味方ユニットの攻撃処理（当たり判定・ダメージ計算）
- アニメーションと攻撃判定の同期処理
- ダメージ数値の描画
- カメラズーム演出
- 既存コードの改修・バグ修正

## 技術的なポイント
- アニメーションのフレームに応じて攻撃判定を発生させる同期処理
- ダメージ数値を個別に管理し、同時ヒット時でも視認性を確保
- 描画スケーリングによるカメラズーム機能の実装

## 工夫した点
攻撃のタイミングとアニメーションを一致させることで、プレイヤーに違和感を与えない設計にしました。  
また、ゲーム開始時や終了時にカメラズームを用いた演出を加え、視覚的な分かりやすさと演出効果を向上させています。

## 学んだこと
チーム開発において、他人のコードを理解しながら改修する力の重要性を学びました。
